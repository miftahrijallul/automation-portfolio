# LaTeX Layout Samples

Anonymized excerpts (first 3 pages each) from real journal-article layouts I produced as technical staff for two university journals. Author names, affiliations, journal identity, DOIs, and contact details have been replaced; the underlying articles are published under CC BY 4.0.

| Sample | Shows |
|---|---|
| [sample-1.pdf](sample-1.pdf) | Two-column research-article layout: title page, abstract/keyword panels, drop caps, linked citations, ORCID marks, figure placement |
| [sample-2.pdf](sample-2.pdf) | Dense methods/results layout: numbered subsections, tables, running heads, footer compliance marks |

Both are compiled from a custom in-house LaTeX style I maintain (XeLaTeX, custom fonts, automated from DOCX manuscripts via a Python pipeline). Full documents and template-compliance work available on request during an active order.
